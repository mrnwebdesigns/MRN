# MRN Disable Comments

Standalone WordPress plugin version of the hardened comments-disable logic.

## Version
- 1.2.0

## What it does
- Blocks comment submission with HTTP 403.
- Forces comments/pings closed across post types.
- Removes comments UI from admin menus/admin bar/meta boxes/dashboard.
- Blocks direct access to comment admin screens.
- Removes WordPress REST comment endpoints.
- Disables comment feeds.
- Removes XML-RPC comment and pingback methods.
